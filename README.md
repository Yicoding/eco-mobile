# eco-mobile

基于`react`的移动端扩展组件

[查看文档和示例][site]

## 使用

### npm 或 yarn 安装

```shell
npm install eco-mobile --save
```

<br />

```shell
yarn add eco-mobile
```

- 自定义配置的 `webpack` 项目，请安装 [babel-plugin-import] ，将上面配置添加到 `babel` 的 `plugins` 中。
