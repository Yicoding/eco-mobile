## API

### DrawerModal

| 属性     | 说明                                         | 类型                           | 默认值  |
| -------- | -------------------------------------------- | ------------------------------ | ------- |
| title    | 标题                                         | `string or React.ReactElement` | `-`     |
| anchor   | 出现的位置，One of: left, right, top, bottom | `enum`                         | `false` |
| onCancel | 点击背景触发关闭事件                         | `() => void`                   | `-`     |
| anchor   | 出现的位置，One of: left, right, top, bottom | `enum`                         | `false` |
| visible  | 对话框是否可见                               | `boolean`                      | `false` |
